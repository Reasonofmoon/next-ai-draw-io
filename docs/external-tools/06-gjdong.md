# gjdong (표준주소실록)

> ⚪ **참고만** · EdTech 도메인과 무관

주소 데이터 한방 정규화.

## 대상 / 접근
- **URL**: https://gjdong.vercel.app
- **GitHub**: https://github.com/chrisryugj/gjdong

## 기능
- 엑셀 일괄 변환 (CSV/XLSX)
- 지도 번호 마커 (최단 동선)
- Tableau Geocoder
- Chrome 익스텐션 (Plasmo)

## 참고할 지점

1. **Plasmo 기반 Chrome 확장** — 우리도 장기적으로 "교사가 온라인 영어 지문 아무 사이트에서 우클릭 → 이 앱으로 보내기" 같은 확장 기능을 고려할 때 Plasmo가 후보.
2. **엑셀 일괄 처리 UX** — 교사가 문제지 메타데이터(학년/단원/유형)를 엑셀로 관리하는 경우가 많음. 엑셀 드래그드롭 → 일괄 처리 패턴 참고.

그 외 EdTech 로드맵과 직접 접점 없음.
