# HWP 배치 데모 가이드

이 프로젝트는 `HWP/HWPX 업로드 → 영어 지문 감지 → draw.io 다이어그램 자동 생성 → PNG 렌더링 → HWP 재삽입` 흐름을 한 화면에서 시연할 수 있습니다.

데모 진입 주소는 `http://localhost:6002/test-hwp`입니다. 로케일 미들웨어가 자동으로 `/en/test-hwp`로 연결합니다.

## 1. 데모 실행

가장 쉬운 방법은 아래 둘 중 하나입니다.

```bat
demo-hwp-batch.bat
```

```bash
npm run demo:hwp
```

실행 스크립트가 하는 일:

- `.env.local` 존재 여부 확인
- `node_modules`가 없으면 `npm install` 실행
- 개발 서버를 새 PowerShell 창에서 실행
- 브라우저에서 `http://localhost:6002/test-hwp` 열기

## 2. 사전 준비

`.env.local`에는 적어도 하나의 AI 공급자 설정이 필요합니다. 가장 간단한 예시는 아래와 같습니다.

빠르게 시작하려면 아래 파일을 복사하면 됩니다.

```powershell
Copy-Item env.hwp-demo.example .env.local
```

```env
AI_PROVIDER=openai
AI_MODEL=gpt-4.1-mini
OPENAI_API_KEY=sk-...
```

발표 전에 확인할 것:

- 인터넷 연결
- `.hwp` 또는 `.hwpx` 샘플 파일 1개
- AI API 키
- 포트 `6002`가 비어 있는지 여부

## 3. 추천 시연 순서

### A. 30초 예열 데모

`🎨 디자인 미리보기 (6개 샘플)` 섹션에서 `✨ 6개 샘플 돌려보기`를 누릅니다.

이 단계는 HWP 업로드 없이도 아래를 빠르게 보여줍니다.

- 질문 유형별 다이어그램 스타일
- draw.io XML 자동 생성
- PNG 렌더링 결과
- draw.io 공유 링크

### B. 메인 데모: HWP 배치 자동 생성 + 삽입

1. `📄 HWP 파일 선택`으로 수능형 영어 지문이 들어 있는 `.hwp` 또는 `.hwpx`를 업로드합니다.
2. 하단 `📖 감지된 지문` 카드에서 문제 번호, 유형, 삽입 위치를 확인합니다.
3. 필요하면 상단 `Width(px)` / `Height(px)`를 조절합니다.
4. `🤖 전체 AI 파이프라인 (지문 → HWP)` 섹션에서 `🚀 N개 지문 파이프라인 실행`을 누릅니다.
5. 진행 상태가 `AI → PNG → inserting` 순서로 바뀌는 것을 보여줍니다.
6. 완료되면 다이어그램이 삽입된 새 HWP가 자동 다운로드됩니다.
7. 원하면 `📦 ZIP 다운로드`로 PNG 묶음도 함께 내려받습니다.

이 흐름이 시연의 핵심입니다. 즉, 한글 문서에서 지문을 읽고, draw.io 다이어그램을 여러 개 만들고, 다시 HWP에 넣는 배치 자동화가 한 번에 끝납니다.

## 4. 보조 데모 시나리오

### 특정 지문만 바로 보여주고 싶을 때

`✏️ 지문 직접 붙여넣기 (1개 end-to-end)` 섹션을 사용합니다.

- 영어 지문을 직접 붙여넣기
- 유형 선택
- AI 생성부터 HWP 삽입까지 즉시 실행

### 이미 있는 PNG를 HWP에 반복 삽입하고 싶을 때

`📎 PNG → HWP 직접 삽입` 섹션을 사용합니다.

- PNG 파일 1개 선택
- 특정 위치 삽입
- 또는 감지된 모든 지문 위치에 일괄 삽입

이 모드는 "AI 생성 없이도 HWP 재삽입 자체는 된다"는 백업 시연용으로 좋습니다.

## 5. 발표 멘트 예시

"이 화면에서 HWPX 학습지를 올리면 영어 지문을 자동 감지합니다. 그다음 각 지문을 AI가 draw.io XML로 구조화하고, PNG로 렌더링한 뒤, 다시 한글 문서에 자동 삽입합니다. 즉 교사는 HWP를 올리고 결과 HWP를 내려받기만 하면 됩니다."

## 6. 문제 발생 시 체크 포인트

- 업로드는 되는데 생성이 안 되면 `.env.local`의 AI 키를 확인합니다.
- 페이지가 안 열리면 `6002` 포트에서 서버가 떴는지 확인합니다.
- 지문 감지가 약하면 다른 샘플 파일로 바꾸거나 `지문 직접 붙여넣기`로 우회합니다.
- 삽입 위치가 어긋나면 감지 카드의 `삽입 위치 지정` 버튼으로 section/paragraph를 다시 맞춥니다.
